Why Vim?

- Lightweight
- Makes some editing really easy
- Everywhere
- Remote coding via ssh
- Pair programming (ssh + tmux)
- Important skill (like regex)
- It's important to 'Feel Stupid', Learn Somthing

"Vim is like medicine, if used wrong you can kill people with it and there's 
always a better doctor than you somewhere"

Vim commands:
  - h,j,k,l
  - move to line number
  - H, L, M (high, low, medium)
  - zz
  - searching
  - f: inclusive search in line
  - t: exclusive search in line
  - viw : highlight word cursor is on
  - diw : delete word cursor is on
  - c-v : vertical visual block
  - a : append text after cursor
  - A : append text at end of the line
  - i : insert text before cursor
  - I : insert text before first non-blank in line
  - s : erase the current letter under the cursor, set insert-mode
  - S : Erase the whole line, set insert-mode
  - "0 : most recent yanked text
  - C-b: C-f, C-d, C-u
